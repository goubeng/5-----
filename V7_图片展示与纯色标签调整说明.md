# V7 调整说明

本版在 V6“恢复原排版并同步文案”的基础上，只调整两类样式，不改变原页面结构：

## 1. 图片展示方式调整

场景页中的图片读取方式已调整为：

- 图片宽度：100%
- 图片高度：按图片原始比例自动撑开
- 图片不裁切、不拉伸、不变形
- 图片路径继续使用：
  - `images/model1/1.png`
  - `images/model2/1.png`
  - `images/model3/1.png`
  - `images/model4/1.png`

注意：HTML 中使用 `/`，不是 Windows 路径中的 `\`。  
例如应使用 `images/model1/1.png`，不是 `images\model1\1.png`。

## 2. 渐变标签改为纯色

已将所有页面中的渐变标签、流程当前节点、编号圆标、建设价值标签等改为统一纯色棕金底：

- 纯色底：`#B56A25`
- 边框色：`#C99548`
- 文字色：`#FFFFFF`

保持原来的圆角、阴影、大小和排版不变，只去掉渐变效果。
